## 想定情報
店舗の特徴
・食事中心の居酒屋
・ランチ（11：00～14：00）は定食、弁当のみ（弁当は日替わり1種のみ限定20食）
・夜は居酒屋として営業（17；00～22；00）
・カウンター8名、個室4名×4、個室6名×2（予約は最大6名まで）
→騒がしくならず落ち着いて料理とお酒を楽しんでもらうため
・健康志向の野菜を多めに使った料理が特徴（その分少し高めの値段設定）
ターゲットユーザー
・30～50代独身（男性中心）
・50代～シニア層
・健康も意識しており野菜の入った料理が食べたい
・落ち着いて食事がしたい
ユーザーに与えたい印象
・落ち着いた雰囲気（騒がしくなさそう）、美味しそうで健康的な食事
・ランチ→1000円前後でバランスの良い食事
・夜→同僚ともしくはひとりで食事とお酒をのんびりと
・サイトを見る→場所や営業時間をすぐに確認できる→予約が簡単にできる
→気になった時にすぐに情報にアクセス出来るようにする
・要望、感想を気軽に出すことができる
・個室で落ち着いて会話ができる
・スッキリしていて見やすい（文字少なく大き目、写真が多くなり雑多な印象を与えないようにする）
・『とりあえず一度行ってみよう。』となることが理想
Webページイメージカラー
ベース：ベージュ（淡く柔らかい色）
メイン；黒（コントラストを強く落ち着いた雰囲気に）
アクセント：赤（強調色）
